**Digital Distraction \& Academic Performance Analyzer**



**Project Overview**



This project analyzes the impact of digital distraction, sleep patterns and study habits on student productivity and academic performance using Python, SQL and Power BI.



**Technologies Used**



\* Python

\* SQL

\* Power BI

\* Pandas

\* Excel/CSV



**Features**



\* Productivity Score Analysis

\* Stress Level Analysis

\* Mobile Usage Analysis

\* Study Hours vs Productivity Analysis

\* Interactive Power BI Dashboard



**Dashboard Insights**



\* Average Productivity Score

\* Stress Level Distribution

\* Mobile Usage Comparison

\* Productivity Trend Visualization



**Tools Used**



\* MySQL Workbench

\* Python Pandas

\* Power BI Desktop

\* Excel
